Module 4 - install guide
========================

Required software
-----------------
- Node.js LTS
- text editor or IDE
- Git if the project is cloned from a repository

Install and run
---------------
1. Open a terminal in `moodul4/backend`
2. Run `npm install`
3. Copy `.env.example` to `.env`
4. If needed, change `ADMIN_EMAIL`, `ADMIN_PASSWORD`, `SESSION_SECRET`, `PORT`
5. Run `npm run dev`

Default address
---------------
`http://localhost:3005`

Routes to check in browser
--------------------------
- Home page: `http://localhost:3005/`
- Store: `http://localhost:3005/shop`
- Product detail: `http://localhost:3005/product/1`
- Contact: `http://localhost:3005/contact`
- Admin login: `http://localhost:3005/admin/login`
- Saved contacts: `http://localhost:3005/admin/contacts`

Static Module 3 site
--------------------
If Module 3 is opened through a local HTTP server, `contact.html` can submit to:
`http://localhost:3005/api/contact`
