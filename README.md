### Module 3 - Static website
Folder: `moodul3/`

Pages:
- `index.html` - team landing page
- `shop.html` - store with filters and sorting
- `product.html` - single product view
- `contact.html` - validated contact form

### Module 4 - Custom backend
Folder: `moodul4/backend/`

Stack and features:
- Node.js + Express + EJS
- file-based `db.json`
- public home page, store, product detail, contact form
- admin login
- product CRUD
- contact message storage

Documentation:
- `moodul4/dokumentatsioon/dokumentatsioon.md`
- `moodul4/installijuhend.txt`

## Running the backend

```bash
cd moodul4/backend
npm install
copy .env.example .env
npm run dev
```

Default local address: `http://localhost:3005`
